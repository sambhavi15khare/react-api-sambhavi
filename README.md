This assignment demonstrates the use of Custom React Hooks and API Fetching in a React application.  
The project is built using Create React App and includes a reusable custom hook (useFetch.js) to fetch and display data dynamically.


 Live Demo (Deployed Link)
 https://sambhavi15khare.github.io/assignment5-sambhavi-khare/

 Features Implemented
- Custom Hook: useFetch() 
- Fetching & displaying data from a public API
- Functional components
- Clean, modular code structure
- Conditional rendering (loading, error, success states)
- Responsive basic UI
